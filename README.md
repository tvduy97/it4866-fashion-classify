Các lệnh liên quan:

Cài Django:
```
python -m pip install Django
```

Cài Tensorflow:
```
python -m pip install tensorflow==2.0.0-alpha0
```
Cài Pilow:
```
python -m pip install Pillow
```

Tạo Django project:
```
django-admin startproject fashion_classify
```

Run server:
```
cd fashion_classify
python manage.py runserver
```

Create app:
```
python manage.py startapp myapp
```
